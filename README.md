1. Start the Backend Server (API)
Open a terminal in the project root and run:

powershell
cd "c:\report project\server"
npm start
You should see: ✓ FORCED ZERO-FAIL LOCAL MODE (Active)

2. Start the Frontend (Vite)
Open a second terminal and run:

powershell
cd "c:\report project\client"
npm run dev
3. Open the Application
Navigate to the following URL in your browser: http://localhost:5173
